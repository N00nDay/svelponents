# Skeleton
## Basic

```html
<script>
  import { Skeleton } from "svelponents";
</script>

<Skeleton />
```

## Paragraphs and lines

```html
<script>
  import { Skeleton } from "svelponents";
</script>

<Skeleton paragraph={2} lines={3} />
```

## Avatar

```html
<script>
  import { Skeleton } from "svelponents";
</script>

<Skeleton avatar />
```
