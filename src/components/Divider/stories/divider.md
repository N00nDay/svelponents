# Divider
## Horizontal

```html
<script>
  import { Divider } from "svelponents";
</script>

<LoremIpsum />
<Divider />
<LoremIpsum />
<Divider />
<LoremIpsum />
```

## With title and align

```html
<script>
  import { Divider } from "svelponents";
</script>

<LoremIpsum />
<Divider>Text</Divider>
<LoremIpsum />
<Divider align="left">Left Text</Divider>
<LoremIpsum />
<Divider align="right">Right Text</Divider>
<LoremIpsum />
```
